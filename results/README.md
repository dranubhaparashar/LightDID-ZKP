# Results folder

This folder contains the results included with the repository.

## `summary/`

Container-baseline benchmark summaries from the LightDID-ZKP manuscript:

- `bbs_50run_summary.csv`: BBS selective-disclosure benchmark summary for P2.
- `anoncreds_50run_summary.csv`: AnonCreds credential-backed predicate benchmark summary for P4.
- `scaling_ratios.csv`: derived 4-to-64 attribute scaling ratios.
- `per_attribute_growth.csv`: derived approximate per-attribute growth.

## `selector/`

Generated by `python experiments/selector_demo.py`.

## `ablation/`

Generated by `python experiments/ablation_study.py`. Shows why unsafe cost-first selection can violate predicate semantics.

## `resource_sensitivity/`

Generated by `python experiments/resource_sensitivity.py`.

## `figures/`

Generated by `python experiments/make_figures.py`.

## Claim boundary

The summary CSVs characterize the reported container-baseline environment. They must not be described as Android, Raspberry Pi, microcontroller, CPU-energy, or physical IoT-device results unless new measurements are added from those platforms.
